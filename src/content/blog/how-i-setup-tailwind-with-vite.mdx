## ğŸ“ 2. Configure `tailwind.config.js`

```js
/** @type {import('tailwindcss').Config} */
export default {
  content: ['./index.html', './src/**/*.{js,ts,jsx,tsx}'],
  darkMode: 'class',
  theme: {
    extend: {},
  },
  plugins: [],
}
```

This sets up file scanning and enables the `class`-based dark mode strategy.

## ğŸ¨ 3. Add Tailwind to your CSS

In `src/styles/index.css` (or similar), add:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

Then import this CSS into your entry point, usually in `main.tsx`:

```ts
import './styles/index.css';
```

## ğŸŒ™ 4. Enable Dark Mode

I use a custom `ThemeProvider` and a toggle button component. Tailwind automatically applies `.dark` styles using the `class` strategy:

```tsx
<div className="bg-white text-black dark:bg-zinc-900 dark:text-white">
  ...
</div>
```

This allows theme switching without using the OS preference.
