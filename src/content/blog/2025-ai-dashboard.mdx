The **AI Dashboard** project is my latest portfolio addition — a real-time, AI-powered analytics tool designed to turn raw data into actionable insights.  
You can view the live version here: [AI Dashboard Live](https://ai-dashboard-1acrwhndz-andrewteeces-projects.vercel.app/).

---

## 🛠️ Tech Stack

- **Next.js 15** — App Router, server components, and edge-ready architecture  
- **React 19** — streamlined UI with concurrent features  
- **Tailwind CSS** — fully responsive, custom-themed design system  
- **OpenAI API** — intelligent text analysis and AI-powered recommendations  
- **Vercel** — fast, zero-config deployment with analytics  
- **Framer Motion** — smooth animations for transitions and component states  

---

## ✨ Highlights

- **Real-Time AI Insights** — integrates OpenAI to summarize, predict, and recommend actions instantly  
- **Customizable Dashboard Widgets** — drag-and-drop components to tailor your workspace  
- **Responsive & Accessible UI** — works seamlessly across desktop and mobile  
- **Data Visualization** — interactive charts and graphs for clearer decision-making  
- **Performance-Optimized** — Core Web Vitals-friendly with image optimization and code splitting

## 📈 Current Status

- **MVP Complete** — core AI features, charts, and user interface are implemented  
- **Live Demo** — deployed to Vercel with a working production build  
- **Next Steps** — add authentication, persistent user settings, and expanded AI tools for deeper analytics

## 🎯 Why This Project?

The AI Dashboard is designed for business owners, analysts, and developers who need a **centralized, intelligent hub** for tracking performance. It’s a proof of concept for blending **modern React architecture** with **cutting-edge AI** to deliver both speed and insight in a single, beautiful interface.
